### Track A — Inverse Problems & Parameter Identification: Recovering Burgers' Equation Viscosity via Physics-Informed Neural Networks

This project uses a Physics-Informed Neural Network to recover the unknown viscosity parameter ν in the nonlinear Burgers' equation from just 200 sparse, noisy observations of the solution, treating ν as a trainable parameter optimized jointly with the network alongside the PDE residual, initial, and boundary condition losses. An initial implementation converged but recovered the wrong ν (75-215% error) regardless of optimizer tuning; the root cause was traced to an identifiability problem, ν this small produces a viscous shock too thin for the sampled observations to ever land inside, and was fixed by sampling observations and collocation points with deliberate coverage of the shock region, combined with a log-space parametrization of ν and L-BFGS refinement after Adam. The final model recovers ν to within 0.64% error on clean data and under 4% error even at 10% observation noise, while a classical scipy curve-fitting baseline (which assumes a linear-diffusion solution shape) fails with 1184% error, demonstrating a case where PINNs succeed precisely because they require no closed-form assumption about the nonlinear solution's shape.
